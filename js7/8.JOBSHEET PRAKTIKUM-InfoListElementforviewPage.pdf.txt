Pertemuan 8 – Implementasi Info List (View Page) di Filament 
 
       Capaian Pembelajaran 
Setelah mengikuti praktikum ini, mahasiswa mampu: 
1. Memahami konsep Info List pada Filament 
2. Mengubah tampilan View Page dari form menjadi display informasi 
3. Menggunakan TextEntry, ImageEntry, dan IconEntry 
4. Menggunakan Badge, Color, Icon, dan Format Date 
5. Mendesain halaman detail (show page) yang lebih profesional 
Framework yang digunakan: 
Filament 
 
     A. Latar Belakang 
Pada pertemuan sebelumnya, kita telah membuat Wizard Form pada module Product. 
Namun ketika tombol View diklik, halaman detail masih menampilkan form input (kurang tepat 
untuk tampilan informasi). 
Solusinya adalah menggunakan Info List agar data ditampilkan dalam bentuk informasi (read-only 
display). 
 
     B. Konsep Info List 
Info List digunakan untuk: 
• Menampilkan data detail record 
• Mengganti tampilan input menjadi display-only 
• Membuat halaman View lebih profesional 
Komponen Info List mirip dengan: 
Form Table Info List 
TextInput TextColumn TextEntry 
FileUpload ImageColumn ImageEntry 
Checkbox IconColumn IconEntry 

 
     C. Mengedit Product Info List 
Buka file: 
• ProductInfolist.php 
 
 
     D. Membuat Section – Product Info 
Section::make('Product Info') 
    ->schema([ 
        TextEntry::make('name') 
            ->label('Product Name') 
            ->weight('bold') 
            ->color('primary'), 
 
        TextEntry::make('id') 
            ->label('Product ID'), 
 
        TextEntry::make('sku') 
            ->label('Product SKU') 
            ->badge() 
            ->color('success'), 
 
        TextEntry::make('description') 
            ->label('Product Description'), 
    ]) 
    ->columnSpanFull(), 

 
 
  Tampilan Section Product Info 

 
 
 
 
 
     E. Section – Pricing & Stock 
Section::make('Pricing & Stock') 
    ->schema([ 
        TextEntry::make('price') 
            ->label('Product Price') 
            ->icon('heroicon-o-currency-dollar'), 
 
        TextEntry::make('stock') 
            ->label('Product Stock'), 
    ]), 
 

Hasil: 
• Price memiliki icon dolar 
• Stock ditampilkan dalam format teks 
 
 
     F. Section – Media & Status 
 
  Menampilkan Gambar 
ImageEntry::make('image') 
    ->label('Product Image') 
    ->disk('public'), 

 
 
 

  Menampilkan Status Boolean 
IconEntry::make('is_active') 
    ->label('Is Active') 
    ->boolean(), 
 
IconEntry::make('is_featured') 
    ->label('Is Featured') 
    ->boolean(), 
 
Jika true → icon check 
Jika false → icon silang 

 
 
  Menampilkan Tanggal dengan Format 
TextEntry::make('created_at') 
    ->label('Product Creation Date') 
    ->date('d M Y') 
    ->color('info'),

     H. Ringkasan Komponen Info List
